rails/heroku エラー「ECONNREFUSED: connect ECONNREFUSED」

Rails・herokuのエラー

Railsチュートリアルをやっていて、heroku run rails db:migrateをしようとしたところ、


$ heroku run rails db:migrate
ECONNREFUSED: connect ECONNREFUSED 50.19.103.36:5000

というエラーが出ました。
調べてもイマイチ解決策が出ず。。。

解決

「ECONNREFUSED」というのはネットワーク関連のエラーで、つまりホストがサービスの接続や転送を拒否した時のメッセージだと思われます。
エラーログを見ると:5000とありますが、これはおそらく接続ポート番号のことで使っているネットワークのファイアウォールの設定等により5000番ポートが使えず接続できないということを言ってるのかと思います(たぶん)

自分はこの時学校のインターネットを使っていたため、上のような説明を見たとき「まさかこれが原因か・・・？」とテザリングに変えてみたところ、アッサリ解決しました。

この問題に限らず、ネットワークを変えてみるというのは一つの手かもしれません(わかりません)

これで解決しない場合は、heroku logsを見てみると Error R13とあるので、それを元に調べてみるとこのへん(Stack Overflow)が役に立つかもしれません。
