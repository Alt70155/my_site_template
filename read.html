メジャーなWebサイトでよく見る『2カラムレイアウト』をflexboxで作ってみたいと思います！
めっちゃ簡単にできました。

2カラムレイアウトとは

2カラムレイアウトとは、メインコンテンツ(記事本文など)の右か左の部分にサイドバーとして列を配置する構成の事を言います。(下の画像が例)

記事の横に追加情報やその他記事、広告など様々な情報を置く事ができるので、1ページの情報量が多くなるのが利点かなーと思っています。
他にも3カラムレイアウトというものもあります。

実装

flexboxで簡単に実装できます。それに加えて微調節を行って綺麗に。
追加でメディアクエリを使って、横幅が小さくなってサイドバーがはみ出すタイミングで下に回り込ませています。

HTML

[

ヘッダー






メインコンテンツ


article...





サイドバー






フッター



]

CSS

[
header {
max-width: 980px;
height: 300px;
margin: auto;
border: 1px solid black;
text-align: center;
}

#wrap {
display: flex; / 全体を横並びに /
flex-wrap: wrap;　 / 下に回り込む設定 /
justify-content: center; / flexboxの要素をセンターに寄せる /
margin: auto;
max-width: 980px;
height: auto;
border: 1px solid black;
}

main {
width: 75%; / mainとside-barで3:1に設定 /
height: 1000px;
text-align: center;
background-color: #BAD3FF;
}

.left-bar {
width: 25%;
text-align: center;
background-color: #75A9FF;
}

footer {
max-width: 980px;
height: 250px;
margin: auto;
border: 1px solid black;
text-align: center;
}


@media screen and (max-width: 997px) {
main {
width: 100%;
}

.left-bar {
width: 100%;
height: 300px;
}
}
]

wrapのmax-widthやmainのmin-width値を変えた場合、left-barのwidthやメディアクエのmax-width値も連動して変更する必要があります。
